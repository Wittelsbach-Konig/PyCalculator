# PyCalculator
### Description
A Calculator project developed using python and ANTLR library.

### Step 1
Clone repository:
```bash
git clone git@github.com:Wittelsbach-Konig/PyCalculator.git
```
### Step 2
Create and activate virtual enviroment:
```bash
poetry install
```
```bash
poetry shell
```
### Step 3
Run the calculator:
```bash
python run.py --input input.txt [--debug]
```
